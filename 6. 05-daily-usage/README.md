# 05. Повседневное использование

## Что делать каждый день

- Используйте телефон как основной смартфон
- Делайте фото, видео, звонки
- Пользуйтесь мобильным интернетом
- Установите eSIM, если нужно
- Устанавливайте обычные Android-приложения

**Цель:** показать, что вы реальный пользователь, а не фармер.
