# 24. Отслеживание метрик и прогресса

## Как мониторить свою активность

1. **Seed Vault Wallet** → вкладка **Activity Tracking** (иконка спидометра)
2. Следите за заполнением колец (Daily, Weekly)
3. Проверяйте текущий тир (Scout → Sovereign)
4. Используйте внешние инструменты:
   - Solana Explorer
   - Step Finance
   - Birdeye.so
   - Dune Analytics (дашборды по Seeker)

**Совет:** Ведите личную таблицу (Notion / Google Sheets) с ежедневным количеством транзакций и типами активности.
