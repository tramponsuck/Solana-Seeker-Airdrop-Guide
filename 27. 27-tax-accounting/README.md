# 27. Налоги и учёт

- Airdrop $SKR считается доходом в момент клейма
- Ведите учёт всех транзакций (входящие/исходящие)
- Рекомендуемые сервисы: Koinly, CoinTracker, CryptoTaxCalculator
- Для жителей Украины и СНГ — консультируйтесь с налоговым специалистом
- Сохраняйте скриншоты клеймов и транзакций
