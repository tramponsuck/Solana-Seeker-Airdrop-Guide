# 26. Управление кошельками

- Основной кошелёк — **Seed Vault** (аппаратная защита)
- Можно подключать дополнительные кошельки (Phantom, Solflare)
- Не экспортируйте seed phrase Seed Vault без крайней необходимости
- Используйте разные кошельки для крупных сумм и мелких операций
- Регулярно делайте резервные копии (только в безопасном месте)
