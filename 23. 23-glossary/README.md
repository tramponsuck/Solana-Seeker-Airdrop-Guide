# 23. Глосарій термінів

- **$SKR** — Нативний токен екосистеми Solana Seeker
- **Seed Vault** — Аппаратний гаманець на телефоні
- **Seeker ID** — Унікальний домен .skr
- **Genesis Token** — NFT, що підтверджує володіння пристроєм
- **Sovereign** — Найвищий тир активності
- **Guardians** — Стейкінг пул для $SKR
- **Activity Rings** — Кільця щоденної активності

Цей розділ допомагає новачкам швидко розуміти термінологію.
