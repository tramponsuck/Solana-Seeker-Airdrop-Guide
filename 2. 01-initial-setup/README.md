# 01. Начальная настройка

## Пошаговая инструкция

1. Включите **Usage & Diagnostics**  
   `Settings → Solana Mobile → Usage & Diagnostics`

2. Обновите **Seed Vault** до последней версии через dApp Store

3. Настройте Seed Vault Wallet с биометрией (fingerprint/face)

4. Подключите SIM-карту или eSIM (рекомендуется encryptSIM)

5. Включите мобильный интернет и используйте телефон хотя бы 30–60 минут в день

## Важно
- Не пропускайте этот шаг — без него активность не будет засчитываться.
- Используйте телефон как основной, а не только для airdrop.
