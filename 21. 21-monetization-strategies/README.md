# 21. Стратегії монетизації

## Як заробляти на Solana Seeker

- Отримання $SKR airdrop
- Стейкінг rewards (SOL + $SKR)
- Участь у DePIN проектах (Helium, Grass тощо)
- Продаж/трейдинг NFT, отриманих через активність
- Builder grants для розробників
- Реферальні програми Solana Mobile

**Мета:** Комбінувати кілька джерел доходу, а не тільки чекати airdrop.
