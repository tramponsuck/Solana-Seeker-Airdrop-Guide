# 11. Поширені помилки та як їх уникнути

## ТОП помилок фармерів

- Спам транзакціями (занадто багато однакових дій)
- Використання ботов і скриптів
- Ігнорування Daily Use (тільки onchain)
- Продаж телефону відразу після клейму
- Використання одного типу активності (тільки свопи)

**Правило:** Краще 8 якісних транзакцій, ніж 50 спаму.
