# 25. Советы по железу и оптимизации

- Оптимизируйте расход батареи (включите тёмную тему, ограничьте фоновые процессы)
- Не допускайте сильного перегрева при майнинге ORE
- Регулярно обновляйте систему и Seed Vault
- Используйте хороший чехол и защитное стекло
- Рекомендуется иметь power bank 10 000+ мАч для активного использования

**Производительность:** Seeker хорошо справляется с множеством dApp одновременно.
