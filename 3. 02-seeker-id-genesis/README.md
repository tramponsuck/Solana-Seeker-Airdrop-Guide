# 02. Seeker ID & Genesis Token

## Что нужно сделать

1. Откройте **Seed Vault**
2. Claim **Seeker ID** (ваш уникальный .skr домен)
3. Claim **Genesis Token** (NFT-подтверждение владения устройством)

## Почему это важно

- Genesis Token даёт доступ к эксклюзивным airdrop'ам и перкам
- Seeker ID используется в многих проектах экосистемы
- Без этих двух пунктов часть наград может быть недоступна
